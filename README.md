This project was developed as the final assignment for the Data Analyst course at Vilnius Coding School. The goal was to analyze data and present insights through an interactive and visually consistent Power BI dashboard, focusing on clear KPIs, analytical storytelling, and business-oriented insights.

The project was evaluated with a score of 76/82 (9/10), reflecting a high technical and analytical standard.
